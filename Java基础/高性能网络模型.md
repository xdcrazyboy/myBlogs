# 2. I/O模型
## 2.1 概念理论
阻塞、非阻塞 —— 针对的是调用者
- 阻塞调用 ： 调用结果返回之前，当前线程会被挂起。
- 非阻塞调用 ： 在不能立刻得到结果之前，不阻塞当前线程。

同步、异步 —— 针对被调用者

- 同步处理：被调用方得到最终结果之后才返回给调用方
- 异步处理：被调用方先返回应答，然后再计算调用结果，计算完最终结果再通知并返回给调用方。

`recvfrom`函数（经socket接收数据），视为系统调用
